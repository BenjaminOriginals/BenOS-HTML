# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Beta1   | :white_check_mark: |
| Sierra  | :x:                |

## Reporting a Vulnerability

Report any vulnerabilities to us directly by contacting our business email
here: contact@benjaminoriginals.com
We typically get back to you within 24-72 hours and start working on fixing
bugs, vulnerabilities, and other problems as soon as we initially contact you
back. We will then follow up as soon as your request is fulfilled or if we
find it invalid.
